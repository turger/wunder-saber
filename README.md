# 💎 Wunder saber 💎

Wunderdog beat saber leaderboard.

## Getting started
Clone project

Add firebase:
`npm install -g firebase-tools`

Login to firebase:
`firebase login`

add `.env` file to project root, see `.env.dist`

`yarn`

`yarn start`

or

`yarn build && firebase serve`

## Deploy to firebase
`yarn build-fb`
